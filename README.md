Installation:
-
- Docker
- Docker Compose
- Makefile

run build:
-
- make build

run server:
-
- make up

down server:
-
- make down

run command in container:
-
- make command=""

run all tests:
-
- make test

How to run coverage:
-
- make coverage
- open htmlcov/index.html to see html coverage

Run docker attach:
-
- make attach

Run logs:
-
- make logs
